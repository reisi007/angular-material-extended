## Tech Stack

- **Angular v22** (Standalone Components, Signals, Zoneless)
- **Angular Material v22** + CDK (Material 3 Theming)
- **Vitest** for unit tests
- **Nx** Monorepo
- **pnpm** as package manager
- Full **SSR** support
- **Reactive Forms** + native **Signal-API** per component
